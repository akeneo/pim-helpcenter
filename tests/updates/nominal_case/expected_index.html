    What's new in Akeneo PIM 3.0, 3.1 and 3.2?

    Akeneo PIM 3.0, 3.1 and 3.2 deliver additional features and enhancements requested by our beloved customers and partners. The primary development focus of these new versions covers 4 core topics: managing reference entities, easier connectivity, and workflow and performance improvements. We are also introducing a new enterprise feature named Franklin Insights.
